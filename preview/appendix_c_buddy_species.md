Each time you call the generator, it applies a sequence of bitwise operations (shifts, XORs, multiplications with specific constants) that scramble the bits in a predictable way. The scrambled result is both the output (your random number) and the new state (the seed for the next call). The important property: the same seed always produces the same sequence.
